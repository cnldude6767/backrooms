Backrooms

A horror exploration game inspired by the mysterious world of the Backrooms.

Description

Backrooms is a horror game where players explore strange and unsettling environments while trying to survive the dangers hidden within. Navigate endless rooms, discover secrets, avoid entities, and find a way forward through the unknown.
